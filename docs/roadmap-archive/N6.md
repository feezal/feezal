# N6 — Custom element inspectors ✅ implemented

*Archived roadmap item — Near-term Improvements. Open items: [../ROADMAP.md](../ROADMAP.md) · Index: [README.md](README.md)*

An element can declare `inspector: 'feezal-element-<name>-inspector'` in `static get feezal()` to replace the generic attribute form with a fully custom Web Component. `feezal-sidebar-inspector-attributes.js` renders the named inspector, passes the selected element as `.element`, and listens for `feezal-attribute-changed` events dispatched by the inspector. Used by `feezal-element-circle-light`, `feezal-element-circle-climate`, `feezal-element-circle-cover`, and others. Full spec in `docs/element-spec.md` §3.8.
