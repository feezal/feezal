# U54 — Source editor opens at the current view's line ✅ done

*Archived roadmap item — Editor UX. Open items: [../ROADMAP.md](../ROADMAP.md) · Index: [README.md](README.md)*

**✅ Implemented (07/2026)** — `_revealActiveViewLine()` in [feezal-app-editor.js](../../www/src/feezal-app-editor.js), called at the end of `_enterSourceMode()`: scans every `<feezal-view …>` opening tag in the model text and checks its `name` attribute per hit (attribute order not assumed, exact match — view names are unique), then `revealLineNearTop` + cursor on that line. Not found (fresh unsaved view, malformed doc) → silently stays at the top. Entry-time only — there is no change-active-view-while-source-open flow. Unit-tested (attribute order, exact-name matching, silent fallback); TESTING.md source-mode row extended.

Original item:

Switching to the source editor always starts at the top of the document — on a site with many views, finding the view you were just editing means scrolling/searching. **Wanted:** on entering source mode, the editor jumps to the line of the **currently active view**.

**Implementation:** the source editor is Monaco (`_sourceMode` / `loadMonaco`). After the model is set on entry, locate the active view's opening tag in the serialized HTML — match `<feezal-view` whose `name="<this.view>"` attribute follows within the same tag (don't assume attribute order; a simple regex over the model text covers it since view names are unique) — then `editor.revealLineNearTop(line)` and place the cursor on that line. If the view isn't found (fresh unsaved view, malformed doc), fall back silently to the top. Also apply when the active view is *changed while source mode is open* only if that flow exists — otherwise entry-time only.

**Ships with:** TESTING.md source-editor section update (open source view with a non-first view active → editor is scrolled to its `<feezal-view>` line, cursor there; first view / not-found falls back to top).

**Relates:** U23 (source editor, the other Monaco item), source-mode machinery in feezal-app-editor.js.
