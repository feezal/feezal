# E88 — JSON tree viewer (`feezal-element-basic-json`) ✅ implemented

*Archived roadmap item — Element Ecosystem. Open items: [../ROADMAP.md](../ROADMAP.md) · Index: [README.md](README.md)*

Shipped 07/2026. A Basic element rendering a subscribed JSON payload as a collapsible tree — the "what is this device actually publishing" inspector. Hand-rolled recursive Lit renderer (no dependency): toggleable object/array nodes with previews, per-type-coloured primitive rows. **Expand/collapse state is kept per JSON path** in non-reactive Sets so a live-updating payload never snaps the tree shut; new paths follow `expand-depth`. Non-JSON payloads render as the raw string (never an error); a `max-nodes` guard caps materialised rows with a "+N more" hint. Attributes `subscribe`/`message-property`/`expand-depth` (1)/`max-nodes` (500); tree colours exposed as `--feezal-json-*` props defaulting to canonical theme vars. 8 browser tests + smoke; TESTING.md §6 entry.
