# E39 — Splash / FOUC-prevention element (`feezal-element-system-splash`) ✅ implemented

*Archived roadmap item — Element Ecosystem. Open items: [../ROADMAP.md](../ROADMAP.md) · Index: [README.md](README.md)*

Shipped 07/2026. A System pseudo-element: a viewer-only full-screen overlay (`position:fixed; inset:0; z-index:9999`, solid `--feezal-splash-background`) that hides once (1) the connection is up, (2) the DOM is populated, and (3) a **quiet window** (`settle-window`, default 400 ms) of no messages passes — timing the retained-message burst. A parallel `timeout` (default 3 s from connect) plus a hard cap from element-connect guarantee it never hangs. CSS-only spinner after `spinner-delay` (default 1 s, suppressed when a lottie is set); `logo` centered above; `lottie` replaces the spinner via the shared `@feezal/feezal-lottie` `loadLottie()` chunk (fetched only when the attr is set, fire-and-forget so it never blocks the hide). Editor: static chip only. Place-once (first instance wins, rest `console.warn`). Signals ride on `feezal.connection` — a minimal additive `message` re-dispatch was added to `FeezalConnection`. Attributes `settle-window`/`timeout`/`spinner-delay`/`logo`/`lottie`; canonical `--feezal-splash-*` vars. Browser + smoke + full suites green; TESTING.md §6 entry.
