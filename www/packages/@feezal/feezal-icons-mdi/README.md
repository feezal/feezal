# @feezal/feezal-icons-mdi

[Pictogrammers Material Design Icons](https://pictogrammers.com/library/mdi/) (7400+ icons) as a [feezal](https://github.com/feezal/feezal) icon-set package.

Install through the feezal Package Manager; icons then appear in the icon picker under the **mdi** chip and are referenced as `mdi:<name>` — the same names Home Assistant uses:

```
mdi:lightbulb
mdi:sofa
mdi:thermometer
```

Icons render as inline SVG and follow the active theme via `currentColor`.
